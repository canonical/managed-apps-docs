Onboarding journey
====================

Discovery & sizing
------------------
Assess workload mix (training, inference, notebooks), GPU needs, tenancy model.


Architecture validation
-----------------------
Canonical runs a readiness review (security, operability, observability).


Deployment
----------
ARM template execution + bootstrap automation.


Acceptance & baseline metrics
-----------------------------
Establish initial SLIs (API availability, pipeline success rate, job latency).


Operate
-------
Transition to steady-state with proactive patching & scaling.


Continuous improvement
----------------------
Optimize resource utilization, refine notebooks images, evolve pipeline performance.
